Исправление логики индекса соответствия для AI+.

Что изменено:
- индекс соответствия больше не включается принудительно всегда;
- он показывается только если есть конкретный ориентир:
  1) выбраны конкретные компетенции,
  2) указана должность,
  3) выбран fit-профиль,
  4) введён отдельный запрос.
- текст подсказки в блоке AI+ обновлён.

Если у вас уже стоит предыдущая правка по структуре результатов, проще всего просто заменить файл:
pages/projects/[projectId]/results.tsx
